Components

This website is maintained by the BBMC (in OUHSC Pediatrics), with the input of the investigation collaborators. None of the reports contain PHI, or fined-grained area information that could identify a participant or client.

The website is hosted on GitHub Pages. It is built with Jekyll and R Markdown (and heavily relied on the latter when forming the site’s structure). The majority of the static reports are created with knitr, while the interactive reports are created with Shiny; both use R as a foundation.

Historical Archives

To increase transparency and promote reproducibility, this website is assigned a unique DOI (digital object identifier) after each substantial modification. Each major version is archived (and available for download indefinitely) on a third-party site –where we cannot modify it. The DOI of the current version is 10.5281/zenodo.12234.

Code Collaboration

Interested collaborators are welcome to propose changes or submit an issue on our public GitHub repository. If you’re new to GitHub, see the “Collaborating” section half-way down their help site, or post a comment to us below.
